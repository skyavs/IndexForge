Step 1: Choose Method

Equal Weighted

Equal Weighted

Each component has the same weight in the index, providing equal representation regardless of size.

Best for:

  • Reducing concentration risk
  • Small-cap exposure
  • Sector diversification
Equal Weighted

Equal Weighted

Each component has the same weight in the index, providing equal representation regardless of size.

Best for:

  • Reducing concentration risk
  • Small-cap exposure
  • Sector diversification
Price Weighted

Price Weighted

Components are weighted based on their absolute price levels, with higher-priced stocks having more influence.

Best for:

  • Traditional blue-chip indices
  • Price momentum strategies
  • Simple calculation method
User-defined

User-defined

Customize weights or number of shares for each component based on your strategy.

Best for:

  • Custom investment strategies
  • Factor-based weighting
  • Thematic investing
Market Cap Weighted (coming soon)
This feature will be available in the next update. Components will be weighted according to their market capitalization.

Step 2: Method Parameters

The divisor is used to normalize the index value and maintain continuity when there are changes in the index composition or corporate actions. Choose between a fixed value or a reference date.
Value

Index Divisor

The divisor ensures that only relative changes in the index components affect the index value, not absolute price changes.

When to adjust:

  • Corporate actions (splits, dividends)
  • Component changes
  • Rebalancing events
Value

Fixed Value Divisor

Set a specific numerical value as the divisor. This approach is simpler and provides direct control over the initial index value.

Best for:

  • New indices starting from a specific base value
  • Simple index calculations
  • Direct control over initial index level
Normalised to Date

Date-based Normalization

Use a specific historical date's value as the base. This method is useful for comparing performance over time or aligning with existing indices.

Best for:

  • Historical performance comparison
  • Benchmark alignment
  • Back-testing strategies
